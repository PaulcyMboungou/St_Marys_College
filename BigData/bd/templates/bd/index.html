OBJECTIVE

     BIG DATA is being generated by everything around us at all times. Every digital process and social media exchange produces it. Systems, sensors and mobile devices transmit it. Big data is arriving from multiple sources at an alarming velocity, volume and variety. To extract meaningful value from big data, you need optimal processing power, analytics capabilities and skills.

The objectif of this conference is to provide a forum researchres, educators, student to exchange ideas, to communicate and discuss about reseach finding and new advances in BIG DATA.

SUB THEMES

  • Foundational Models for big data
  • Alogorithms and Programming Techniques for Big Data Processing
  • Cloud Computing Techniques for Big Data
  • Big Data as a service
  • Management issues of social network Big Data
  • Privacy perserving Big Data analytics
  • Big Data encryption
  • Big Data toolkits
  • Scientifics application of Big Data
  • Big Data Economic
